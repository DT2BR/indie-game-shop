About the Game

Flappy Bird is a simple yet addictive arcade game where you guide a small bird through a series of pipes. Each tap on the spacebar makes the bird flap upward — the goal is to survive as long as possible.

Gameplay Preview

The game was recreated as part of this Indie Game Shop project to demonstrate simple game mechanics using HTML5 Canvas and JavaScript.
